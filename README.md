# 🔘 9 Dot Animated Navigation UI (HTML, CSS & JavaScript)

A modern and interactive **9-dot navigation menu UI** built using **HTML, CSS, and Vanilla JavaScript**.

This project showcases a compact navigation component that expands into a **grid-based menu with smooth animations**. It is designed as a **creative frontend UI element** that enhances user interaction and visual experience.

---

## 🧱 Project Structure

```bash
9-dot-animated-navigation/
│
├── index.html         # Structure of navigation menu
├── styles.css         # Styling and animations
├── app.js             # Toggle functionality
├── LICENCE
└── README.md          # Project documentation
```

---

## ✨ Features

### 🔘 Expandable Navigation Menu

* Compact **9-dot layout**
* Expands into a **grid-style menu**
* Smooth transition animation on click

### 🎯 Interactive UI

* Click to toggle menu open/close
* Hover effects on icons
* Clean and minimal user interaction

### 🎨 Modern Design

* Grid-based icon placement
* Animated scaling and transformation
* Visually appealing layout

### ⚡ Lightweight Implementation

* No frameworks used
* Pure **HTML, CSS, and JavaScript**
* Fast and responsive behavior
